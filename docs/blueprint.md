# **App Name**: EHW Lessons

## Core Features:

- Lesson Scheduling: Allow customers to schedule therapy or regular riding lessons, selecting instructor, horse, and time slot.
- Instructor Profiles: Display profiles for each instructor, detailing their expertise and availability.
- Horse Profiles: Showcase the 16 horses with descriptions, photos, and suitability for different lesson types.
- Payment Processing: Securely process lesson payments through the app.
- Schedule Management: Enable administrators to manage instructors' and horses' schedules.
- Admin System: Provide an admin interface to manage horses (availability, health) and instructors (availability).

## Style Guidelines:

- Primary color: Black (#000000), providing a sleek and professional look.
- Background color: White (#FFFFFF), creating a clean and inviting space.
- Accent color: Blue (#007BFF), offering a feeling of trust and stability.
- Font pairing: 'Belleza' for headlines (sans-serif, fashion, art) and 'Alegreya' for body text (serif, elegant).
- Use horse-themed icons for lesson types, instructors, and horse profiles, in black, white, and blue.
- Clean, calendar-based layout for easy scheduling and overview of appointments, mimicking a booksly style.
- Subtle animations on lesson bookings and profile transitions.
